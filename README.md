# Cookie Clicker Game

A simple and fun Cookie Clicker web game. Click the cookie, buy upgrades, unlock achievements, and watch your cookies grow!

## Features
- Cookie clicking for points
- Upgrade store: auto-clickers and multipliers
- Achievements system
- Visual feedback and animations
- Advanced stats (cookies per second)
- Persistent saving/loading

## How to Play
1. Open `index.html` in your browser.
2. Click the cookie to earn cookies.
3. Buy upgrades in the store to increase your cookie production.
4. Unlock achievements as you progress.

## Technologies Used
- HTML
- CSS
- JavaScript
